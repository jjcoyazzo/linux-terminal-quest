# Linux Terminal Quest

Juego interactivo en Bash para aprender comandos básicos de Linux.

## Objetivo
Ayudar a los estudiantes a practicar comandos como:
- ls
- cd
- mkdir
- rm

## Cómo usar
1. Descargar el proyecto
2. Dar permisos:
   chmod +x iniciar_juego.sh
3. Ejecutar:
   ./iniciar_juego.sh
